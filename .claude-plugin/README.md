### プラグイン マニフェストの注意点

`.claude-plugin/plugin.json` を編集する場合、Claude プラグイン バリデータは**未ドキュメントだが厳格な制約**を複数強制するため、`agents: Invalid input` のような曖昧なエラーでインストールが失敗することがある。特に、コンポーネント フィールドは配列である必要があり、`agents` はディレクトリではなく明示的なファイル パスを使う必要があり、`version` フィールドも確実な検証とインストールのために必須となる。

これらの制約は公開例からは分かりにくく、過去にもインストール失敗を繰り返し招いている。詳細は `.claude-plugin/PLUGIN_SCHEMA_NOTES.md` にまとめてあるので、プラグイン マニフェストを変更する前に必ず確認すること。
